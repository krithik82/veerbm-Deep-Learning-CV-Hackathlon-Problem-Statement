To Design Image Based attendance capturing system using Deep Learning and Computer Vision, to train a deep neural network using pytorch

Develped in 

Python Language 3.6 or greater version

Open CV 

Kornia Module used for Face Recognition

Video based attendance capturing system deployed

Auto Encoder used as unsupervised Untrained Network for training and testing

Visual Studio 2015 to be installed to avoid "No module named 'termcolor'"

https://stackoverflow.com/questions/51530437/no-module-named-termcolor/51530570 - refer for solving above issue

https://github.com/ThoughtfulDev/EagleEye/issues/9



The Images 0000000, 0000005 and 0000001 are used for image capturing and identification from the training set provided.

haarcascade_frontalface_default.xml file used to store binary output of image captured
